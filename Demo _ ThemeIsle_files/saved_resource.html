Welcome to Pena Theme

This theme has been inspired by “Living on One Dollar” documentary we watched one night. We were deeply touched and wanted to somehow contribute to the whole project. As theme developers we decided to develop a WordPress theme to help spread the word about it. This theme demo site is built using real content that we hope will raise the awareness about the problems in Peña Blanca and spread the word about this great documentary.
